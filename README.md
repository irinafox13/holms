# Frontend Template

Современный шаблон фронтенда для разработки веб-приложений с использованием передовых инструментов и практик.

## Технологии

- **Node.js** 22.13.1
- **Сборщик**: Vite 6.2.1
- **Препроцессор стилей**: SASS/SCSS
- **Шаблонизатор**: Handlebars
- **JavaScript**: ES6+ с TypeScript для стандартной библиотеки
- **Линтинг**: ESLint + Prettier
- **Валидация**: HTML-валидация
- **Мокирование**: MSW для API запросов

## Структура проекта

```
template/
├── src/
│   ├── assets/           # Статические ресурсы (изображения, шрифты)
│   ├── components.html   # Примеры компонентов
│   ├── data/           # JSON данные для Handlebars
│   ├── main.js         # Точка входа в приложении
│   ├── partials/      # Части Handlebars шаблонов
│   ├── scripts/        # JavaScript модули
│   │   ├── main/      # Основные компоненты приложения
│   │   └── std/       # Стандартная библиотека (TypeScript)
│   └── styles/         # SCSS стили
│       ├── abstracts/   # Переменные, миксины
│       ├── components/  # Стили компонентов
│       ├── modules/     # Стили модулей
│       └── pages/       # Стили страниц
├── dist/               # Сборка проекта
├── mocker/             # Моки для API
└── vite.config.js       # Конфигурация Vite
```

## Установка и запуск

### Требования

- Node.js 22.13.1 или выше
- npm или yarn

### Установка зависимостей

```bash
npm install
# или
yarn install
```

### Скрипты

| Команда                 | Описание                                                                |
| ----------------------- | ----------------------------------------------------------------------- |
| `npm run dev`           | Запуск приложения в режиме разработки с горячей перезагрузкой           |
| `npm run build`         | Сборка приложения для продакшена (минификация, оптимизация, sourcemaps) |
| `npm run preview`       | Запуск локального сервера для просмотра собранного приложения           |
| `npm run lint`          | Проверка и автоматическое исправление ошибок в коде (ESLint)            |
| `npm run format`        | Форматирование кода с использованием Prettier                           |
| `npm run html-validate` | Валидация HTML в папке `dist`                                           |
| `npm run check`         | Полная проверка: сборка, линтинг и валидация HTML                       |
| `npm run prepare`       | Настройка Husky для Git хуков                                           |

## Особенности разработки

### Система сборки

- **Vite** — быстрый сборщик с горячей перезагрузкой
- **SASS** — препроцессор для стилей с переменными и миксинами
- **Handlebars** — шаблонизация с частями и хелперами
- **TypeScript** — строгая типизация для стандартной библиотеки

### Качество кода

- **ESLint** — проверка кода на ошибки и стилистические проблемы
- **Prettier** — единоформатирование кода
- **Husky** — Git хуки для проверки кода перед коммитом
- **HTML Validation** — проверка HTML на соответствие стандартам

### Мокирование API

- **MSW (Mock Service Worker)** — перехват и мокирование HTTP запросов
- Конфигурация через переменные окружения
- Поддержка сложных сценариев тестирования

## Основные зависимости

### Для разработки

- **Vite** — сборщик и dev сервер
- **SASS** — препроцессор стилей
- **ESLint + Prettier** — инструменты качества кода
- **Handlebars** — шаблонизатор

### Для функциональности

- **Axios** — HTTP клиент для запросов

- **Tom Select** — улучшенные селекты
- **Swiper** — слайдеры

## Конфигурация

### Vite

Основная конфигурация в `vite.config.js`:

- Оптимизированная сборка с разделением на чанки
- Поддержка Handlebars шаблонов
- Автоматическое копирование статических ресурсов
- Настройка алиасов для удобного импорта

### TypeScript

Конфигурация в `tsconfig.json`:

- Поддержка JavaScript и TypeScript файлов
- Современные цели компиляции (ES2020)
- Строгая типизация
- Алиасы для всех основных путей проекта
